# TO-DO List Manager (Python)

## Description
A simple **To-Do List Manager** built in Python.  
This project allows users to:

- Add tasks  
- View tasks  
- Delete tasks  
- Exit the program  

It is a **beginner-friendly Python project** that uses lists, loops, and conditional statements.  

---

## Features
1. **Add Task** – Enter a task to save it in the list  
2. **View Tasks** – See all your saved tasks  
3. **Delete Task** – Remove a specific task from the list  
4. **Exit** – Close the program  

---

## How to Run
1. Clone or download the repository  
2. Make sure Python is installed (tested with Python 3.12)  
3. Open terminal / Git Bash in the project folder  
4. Run the program:

```bash
python TODO.py
